###Kaip pasileisti:
1. Make sure you have nodejs v6.11.* `sudo apt-get install nodejs` 
2. Clone repo
3. `cd` to the repo
4. `npm install` to install dependencies
5. `npm start` to start the server
6. `localhost:3000` is the default port
